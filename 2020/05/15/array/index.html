1.两数之和

题目描述：
给定一个整数数组 nums 和一个目标值 target，请你在该数组中找出和为目标值的那 两个整数，并返回他们的数组下标。
示例：
给定 nums = [2, 7, 11, 15], target = 9
因为 nums[0] + nums[1] = 2 + 7 = 9，所以返回 [0, 1]。

  • 暴力法  （两层循环遍历就行，不贴代码了,时间复杂度O(n^2）,空间复杂度O(1))
  • Hash表法  在进行迭代并将元素插入到表中的同时，我们还会回过头来检查表中是否已经存在当前元素所对应的目标元素。如果它存在，那我们已经找到了对应解，并立即将其返回。时间复杂度O(n),只遍历了一次数组,且每个元素只花费了O(1)的时间;空间复杂度O(n), hash表最多N个元素
    public int[] twoSum(int[] nums, int target) {
       if(nums.length==0||nums==null){
           return null;
       }
       int[] ans=new int[2];
       Map<Integer,Integer> map=new HashMap<>();
       for(int i=0;i<nums.length;i++){
           int cur=nums[i];
           int another=target-cur;
           if(map.containsKey(cur)){
               ans[0]=map.get(cur);
               ans[1]=i;
               break;
           }else{
               map.put(another,i);
           }
       }
       return ans;
   }

    2.总持续时间可被60整除的歌曲

    题目描述：
    在歌曲列表中，第 i 首歌曲的持续时间为 time[i] 秒。返回其总持续时间（以秒为单位）可被 60 整除的歌曲对的数量。形式上，我们希望索引的数字 i 和 j 满足 i < j 且有 (time[i] + time[j]) % 60 == 0。
    示例：
    输入：[30,20,150,100,40]
    输出：3
    解释：这三对的总持续时间可被 60 整数：
    (time[0] = 30, time[2] = 150): 总持续时间 180
    (time[1] = 20, time[3] = 100): 总持续时间 120
    (time[1] = 20, time[4] = 40): 总持续时间 60

乍一看题目描述似乎和第一题不咋一样，但是其实实质是一样的，都是求数组中和为目标值的两个数，只不过这道题稍微花里胡哨了一点，把它虚伪的外衣给扒掉，就能看到它光秃秃的内在。首先，总持续时间可以被60整除，那么数组中所有元素取的时候都模60，不就是求两数之和为60，求总共和为60的对数，那么每次匹配到就+1就行。

  • hash表法
    public int numPairsDivisibleBy60(int[] time){
       HashMap<Integer,Integer> map=new HashMap<>();
       int ans=0;
       for(int i=0;i<time.length;i++){
           int cur=time[i]%60;
           if(map.containsKey(cur)) {
               ans+=map.get(cur);
           }
           if(cur==0) {
               map.put(cur,map.getOrDefault(cur,0)+1);
           }
           else {
               int another=60-cur;
               map.put(another,map.getOrDefault(another,0)+1);
           }
       }
       return ans;
   }
  • 数组hash: 由于每首歌的持续时间不可能为负数，也就是数组元素非负，那么就可以用数组来代替hashmap，数组的查询效率高于hashmap，典型的用空间换时间。
    public int numPairsDivisibleBy60(int[] time){
       int ans=0;
       int[] map=new int[60];
       for(int val:time){
           int cur=val%60;
           ans+=map[cur];
           int another=cur==0?0:60-cur;
           map[another]++;
       }
       return ans;
   }
