1.无重复字符的最长子串(#3)

题目描述： 给定一个字符串，请你找出其中不含有重复字符的 最长子串 的长度。
示例：
输入: “abcabcbb”
输出: 3
解释: 因为无重复字符的最长子串是 “abc”，所以其长度为 3。

解法：求的是最长无重复字串，不是子序列，字串是连续的，子序列不一定连续。首先想到的应该是滑动窗口，双指针，快指针指向重复元素时，慢指针应该要指向该元素上次出现的下一个位置，然后快指针再向后遍历，期间要保存长度的最大值。那么怎么判断重复以及快指针遍历过的元素索引该如何保存，这里有两种常见手法。一种就是用hashmap，快指针遍历时将元素以及对应索引存入map；另一种就是用数组。

public int lengthOfLongestSubstring(String s){
        if(s.length()==0||s==null){
            return 0;
        }
        int ans=0;
        int[] index=new int[128];//ascii码一般为7位，也就是128个；扩展ascii为8位
        Arrays.fill(index,-1);//数组初始化全为-1，因为i可以为0
        int i=0,j=0;
        int len=0;
        while(j<s.length()){
            char c=s.charAt(j);
            if(index[c]>=i){ //只要c上个元素的索引在窗口内，也就是[i,j)，c就是重复的
                i=index[c]+1;
                len=j-i;
            }
            index[c]=j++;
            len++;
            ans=Math.max(ans,len);
        }
        return ans;
    }

2.构建回文串检测

题目描述吧：给你一个字符串 s，请你对 s 的子串进行检测。每次检测，待检子串都可以表示为 queries[i] = [left, right, k]。我们可以重新排列子串 s[left], …, s[right]，并从中选择 最多 k 项替换成任何小写英文字母。 如果在上述检测过程中，子串可以变成回文形式的字符串，那么检测结果为 true，否则结果为 false。返回答案数组 answer[]，其中 answer[i] 是第 i 个待检子串 queries[i] 的检测结果。
注意：在替换时，子串中的每个字母都必须作为 独立的 项进行计数，也就是说，如果 s[left..right] = “aaa” 且 k = 2，我们只能替换其中的两个字母。（另外，任何检测都不会修改原始字符串 s，可以认为每次检测都是独立的）
示例：
输入：s = “abcda”, queries = [[3,3,0],[1,2,0],[0,3,1],[0,3,2],[0,4,1]]
输出：[true,false,false,true,true]
解释：
queries[0] : 子串 = “d”，回文。
queries[1] : 子串 = “bc”，不是回文。
queries[2] : 子串 = “abcd”，只替换 1 个字符是变不成回文串的。
queries[3] : 子串 = “abcd”，可以变成回文的 “abba”。 也可以变成 “baab”，先重新排序变成 “bacd”，然后把 “cd” 替换为 “ab”。
queries[4] : 子串 = “abcda”，可以变成回文的 “abcba”。

思路：字串是可以重新排列的，这很关键。因为可以重新排列，所以只需要关心字串各个字符出现的次数就行，出现次数为奇数（实际上应该为1）的字符要小于等于1。修改一次可以消灭掉两个字符，比如abc出现次数为1的字符有3个，你把c改成a，那就只剩下1个b。那么这个问题的关键就在于子字符串字符出现次数的计算。

法一：暴力法超时

int len=queries.length;
       List<Boolean> ans=new ArrayList<>();
       for(int i=0;i<len;i++){

   public boolean isPalindrome(String s,int left,int right,int k){
       String substring=s.substring(left,right+1);
       int[] alpha=new int[26];
       for(char c:substring.toCharArray()){
           alpha[c-97]++;
       }
       int odds=0;
       for(int i=0;i<26;i++){
           if(alpha[i]%2==1){
               odds++;
           }
       }
       return odds-k*2<=1;
   }

法二：将从第一个字符开始的子字符串中各个字符的出现次数都保存在一个二维数组中，中间的子字符串其实也就是两个字串的相减，大大减少了计算的次数。

private int[][] cnt;
   public List<Boolean> canMakePaliQueries(String s, int[][] queries) {
       int len=queries.length;
       count(s);
       List<Boolean> ans=new ArrayList<>();
       for(int i=0;i<len;i++){
           int left=queries[i][0];
           int right=queries[i][1];
           int k=queries[i][2];
           int odds=0;
           for(int j=0;j<26;j++){
                if((cnt[right+1][j]-cnt[left][j])%2==1){
                    odds++;
                }  
           }
           ans.add(odds/2<=k);
       }
       return ans;
   }

   public void count(String s){
       int n=s.length();
       cnt=new int[n+1][26];
       for(int i=0;i<n;i++){
           System.arraycopy(cnt[i],0,cnt[i+1],0,26);
           cnt[i+1][s.charAt(i)-'a']++;
       }
   }

public int findLUSlength(String[] strs){

            @Override
            public int compare(String o1, String o2) {

        for(int i=0;i<strs.length;i++){

    public boolean isSubSequence(String s1,String s2){

5.验证ip地址（#468）

题目描述：编写一个函数来验证输入的字符串是否是有效的 IPv4 或 IPv6 地址。
IPv4 地址由十进制数和点来表示，每个地址包含4个十进制数，其范围为 0 - 255， 用(“.”)分割。比如，172.16.254.1；
同时，IPv4 地址内的数不会以 0 开头。比如，地址 172.16.254.01 是不合法的。
IPv6 地址由8组16进制的数字来表示，每组表示 16 比特。这些组数字通过 (“:”)分割。比如, 2001:0db8:85a3:0000:0000:8a2e:0370:7334 是一个有效的地址。而且，我们可以加入一些以 0 开头的数字，字母可以使用大写，也可以是小写。所以， 2001:db8:85a3:0:0:8A2E:0370:7334 也是一个有效的 IPv6 address地址 (即，忽略 0 开头，忽略大小写)。
然而，我们不能因为某个组的值为 0，而使用一个空的组，以至于出现 (::) 的情况。 比如， 2001:0db8:85a3::8A2E:0370:7334 是无效的 IPv6 地址。
同时，在 IPv6 地址中，多余的 0 也是不被允许的。比如， 02001:0db8:85a3:0000:0000:8a2e:0370:7334 是无效的。

解法：分治法（其实就是最自然的想法，把IP地址按点或分号分割，再进行验证）

public String validIPAddress(String IP){
        if (IP.chars().filter(ch -> ch == '.').count() == 3) {//字符串中点的数量为3
            return validIPv4(IP);
        }
        else if (IP.chars().filter(ch -> ch == ':').count() == 7) {
            return validIPv6(IP);
        }
        else return "Neither";

    }

    public String validIPv4(String ip){

        String[] strs=ip.split("\\.",-1);//limit参数表示模式匹配的次数，limit>0则分割limit-1次；limit为0，分割任意次，但是结尾空字符串丢弃；limit为-1，分割任意次，长度任意
   
        for(String s:strs){
            if(s.length()==0||s.length()>3){
                return "Neither";
            }
            if(s.startsWith("0")&&s.length()>1){
                return "Neither";
            }
            char[] cs=s.toCharArray();
            for(char c:cs){
                if(!Character.isDigit(c)){
                    return "Neither";
                }
            }
            int tmp=Integer.parseInt(s);
            if(tmp>255){
                return "Neither";
            }
        }
        return "IPv4";
    }

    public String validIPv6(String ip){
        String[] strs=ip.split(":",-1);
        for(String s:strs){
            if(s.length()==0||s.length()>4){
                return "Neither";
            }
            char[] cs=s.toCharArray();
            for(char c:cs){
                if(c<48||(c>57&&c<65)||(c>70&&c<97)||c>102){
                    return "Neither";
                }
            }
        }
        return "IPv6";
    }

6.字符串相加（#415）

题目描述：给定两个字符串形式的非负整数 num1 和num2 ，计算它们的和。注意：
num1 和num2 的长度都小于 5100.
num1 和num2 都只包含数字 0-9.
num1 和num2 都不包含任何前导零。
你不能使用任何內建 BigInteger 库， 也不能直接将输入的字符串转换为整数形式。

题解：双指针，从字符串末尾转化为int相加，结果append到stringbuilder中，最后在reverse一下

public String addStrings(String num1, String num2) {
       int len1=num1.length();
       int len2=num2.length();
       int i=len1-1,j=len2-1,carry=0;
       StringBuilder ans=new StringBuilder();
       while(i>=0||j>=0){
           int bit1=i>=0?num1.charAt(i)-'0':0;
           int bit2=j>=0?(int)num2.charAt(j)-'0':0;
           int res=(bit1+bit2+carry)%10;
            carry=(bit1+bit2+carry)/10;
           ans.append(res);
           i--;
           j--;
       }
       if(carry==1){
           ans.append(1);
       }
       return ans.reverse().toString();
   }

7.括号生成(#22)

题目描述：
数字 n 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 有效的 括号组合。
示例：
输入：n = 3
输出：[
“((()))”,
“(()())”,
“(())()”,
“()(())”,
“()()()”
]

题解：递归回溯

public List<String> generateParenthesis(int n) {
       List<String> ans = new ArrayList();
       backtrack(ans, new StringBuilder(), 0, 0, n);
       return ans;
   }

   public void backtrack(List<String> ans, StringBuilder cur, int open, int close, int max){
       if (cur.length() == max * 2) {
           ans.add(cur.toString());
           return;
       }

       if (open < max) {
           cur.append('(');
           backtrack(ans, cur, open+1, close, max);
           cur.deleteCharAt(cur.length() - 1);
       }
       if (close < open) {
           cur.append(')');
           backtrack(ans, cur, open, close+1, max);
           cur.deleteCharAt(cur.length() - 1);
       }
   }

8.多次搜索

题目描述：
给定一个较长字符串big和一个包含较短字符串的数组smalls，设计一个方法，根据smalls中的每一个较短字符串，对big进行搜索。输出smalls中的字符串在big里出现的所有位置positions，其中positions[i]为smalls[i]出现的所有位置。
示例：
输入：
big = “mississippi”
smalls = [“is”,”ppi”,”hi”,”sis”,”i”,”ssippi”]
输出： [[1,4],[8],[],[3],[1,4,7,10],[5]]

题解：字典树匹配前缀，先将字符串中所有字符加入字典树中，然后用大字符串去匹配

//字典树节点结构，主要就是用26个子节点数组代表26个字母
 class Node{
        int id;
        boolean isLeaf;
        Node[] nodes;

        public Node() {
            this.id = -1;
            this.isLeaf = false;
            this.nodes = new Node[26];
        }
    }
    private Node root;
    public int[][] multiSearch(String big,String[] smalls){
        root=new Node();//初始化根节点
        int len=smalls.length;
        List<Integer>[] lists=new List[len];//list数组
        for(int i=0;i<len;i++){
            lists[i]=new ArrayList<>();
        }
        Node p=root;
        for(int i=0;i<smalls.length;i++){
            insertNode(smalls[i],i);//将smalls数组中的字符串插入字典树中
        }
        for(int i=0;i<big.length();i++){
           match(big,i,p,lists);//将big的所有前缀与字典树中的前缀进行匹配
        }
        int[][] ans=new int[len][];//将list数组转化为二维数组
        for(int i=0;i<len;i++){
            ans[i]=new int[lists[i].size()];
            for(int j=0;j<lists[i].size();j++){
                ans[i][j]=lists[i].get(j);
            }
        }
        return ans;
     }

     public void match(String big,int begin,Node root,List<Integer>[] lists){
        char[] cs=big.substring(begin,big.length()).toCharArray();
        for(int i=0;i<cs.length;i++){
            char c=cs[i];
            if(root.nodes[c-'a']==null){//节点为空代表匹配失败
                return ;
            }else{
                root=root.nodes[c-'a'];
                if(root.isLeaf){
                    lists[root.id].add(begin);
                }
            }
        }
        return;
     }

     public void insertNode(String prefix,int id){
        Node p=root;
        char[] cs=prefix.toCharArray();
        for(int i=0;i<cs.length;i++){
            char c=cs[i];
            if(p.nodes[c-'a']==null){
                p.nodes[c-'a']=new Node();
            }
            p=p.nodes[c-'a'];
        }
        p.isLeaf=true;//叶节点
        p.id=id;//叶节点id为smalls对应字符串下标
     }

        Map<String,Integer> word_perfect=new HashMap<>();//放原本字符串
        Map<String,Integer> word_lowercase=new HashMap<>();//放转化为小写后的字符串
        Map<String,Integer> word_vowel=new HashMap<>();//放元音转化后的字符串

        for(int i=len1-1;i>=0;i--){//注意：这里是逆序，为什么是逆序呢，难道正序不行么？是的，正序不行，因为对于转化为小写字母后相等的单词，如果正序，后面的会将前面的索引给覆盖掉，这样后面取到的就不是第一个匹配的索引了
            String word=wordslist[i];
            word_perfect.put(word,i);

            String word_lower=word.toLowerCase();
            word_lowercase.put(word_lower,i);

            String word_convert=convert(word);//这里是将转化为小写字母后的字符串进行元音转化，这样其他字母就不用判断大小写，元音转化也不用考虑大小写
            word_vowel.put(word_convert,i);
        }

        String[] ans=new String[len2];
        for(int i=0;i<queries.length;i++){
            String query=queries[i];
            if(null!=word_perfect.get(query)){
                ans[i]=query;
                continue;
            }
            String lowercase=query.toLowerCase();
            Integer index_lower=word_lowercase.get(lowercase);
            if(null!=index_lower){
                ans[i]=wordslist[index_lower];
                continue;
            }
            Integer index_vowel=word_vowel.get(convert(query));
            if(null!=index_vowel){
                ans[i]=wordslist[index_vowel];
                continue;
            }
            ans[i]="";
        }
        return ans;
    }

    public String convert(String s){
        char[] cs=s.toCharArray();
        String vowel="aeiou";
        for(int i=0;i<cs.length;i++){
            if(vowel.indexOf(cs[i])!=-1){
                cs[i]='#';
            }
        }
        return new String(cs);
    }

10.删除子文件夹（#1233）

题目描述：你是一位系统管理员，手里有一份文件夹列表 folder，你的任务是要删除该列表中的所有 子文件夹，并以任意顺序返回剩下的文件夹。
我们这样定义「子文件夹」：
如果文件夹 folder[i] 位于另一个文件夹 folder[j] 下，那么 folder[i] 就是 folder[j] 的子文件夹。
文件夹的「路径」是由一个或多个按以下格式串联形成的字符串：
/ 后跟一个或者多个小写英文字母。
例如，/leetcode 和 /leetcode/problems 都是有效的路径，而空字符串和 / 不是。
示例：
输入：folder = [“/a”,”/a/b”,”/c/d”,”/c/d/e”,”/c/f”]
输出：[“/a”,”/c/d”,”/c/f”]
解释：”/a/b/“ 是 “/a” 的子文件夹，而 “/c/d/e” 是 “/c/d” 的子文件夹。

public List<String> removeSubfolders(String[] folder) {
    //先将字符串数组中的字符串按照长度升序排序
       Arrays.sort(folder, new Comparator<String>() {
           @Override
           public int compare(String o1, String o2) {
               return o1.length()-o2.length();
           }
       });

       //Treeset是按照排序规则排过序的set，底层是用treemap实现
       //第一次添加的节点为根节点，以后每次添加节点都会根据compare的值来插入
       //如果值为0，则不会插入，值为-1，插入到右节点，值为1，插入到左节点
       Set<String> set=new TreeSet<String>((s1, s2)->{
           if(s1.contains(s2+"/")){//字符串判断是否包含另一字符串的方法
               return 0;
           }
           return s1.compareTo(s2);
       });
       //将字符串数组中的字符串添加到treeset中时，子文件夹不会加入
       Collections.addAll(set,folder);
       return new ArrayList<>(set);
   }
